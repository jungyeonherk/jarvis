⚡ 자연 대화 + 비전 + 웹검색 활성화

💎 무료입니다 (월 1500회)
자비스가 진짜 영화처럼 작동하려면 Google Gemini API 키가 필요합니다.

발급 방법 (3분):
1. https://aistudio.google.com/apikey 접속
2. 구글 계정 로그인
3. "Create API key" 클릭
4. 발급된 키 복사 → 아래 입력란에 붙여넣기
5. 저장 버튼 탭
상태: 키 미설정 (자비스가 정해진 명령만 응답)
🎬 활성화 후 자비스가 할 수 있는 것:
• 자유 대화: "오늘 기분이 어때?" → 자비스가 진짜 응답
• 웹검색: "오늘 비트코인 뉴스 알려줘" → 실시간 검색 후 답변
• 번역: "이거 영어로 뭐야?" → 즉시 번역
• 카메라 비전: "내 얼굴 어때?" → 카메라 보고 답변
• 화면 비전: "이 화면 번역해줘" → 스크린샷 분석
• 보스 정보 기억: 트레이더, 자동매매, 대구 거주 등
• 영국 신사 톤의 자비스 페르소나
⚠️ API 키는 이 폰의 브라우저에만 저장됩니다. 서버로 전송되지 않습니다.
⚠️ 한도 초과 시 24시간 후 자동 리셋 (구글 정책)
